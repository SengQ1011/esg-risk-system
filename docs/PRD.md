# 金融業 ESG 風險評分系統 (ESG-RISK 1.0) - 產品規格書

## 1. 產品概述
本系統解決金融機構在授信與投資決策中，難以量化企業 ESG 風險的痛點。透過 AI 自動化處理非結構化數據，將企業的「風險曝露」與「風險管理」能力轉化為可視化的風險評分。

## 2. 功能規格
| 功能模組 | 描述 | 優先級 |
| :--- | :--- | :--- |
| **數據清洗引擎** | 自動格式化不同來源的雜亂數據，提取產業與關鍵字 | High |
| **AI 評分引擎** | 依據風險模型計算風險得分 (0-100) | High |
| **互動式儀表板** | 呈現評分結果，支援按產業篩選、風險等級預警 | High |
| **決策輔助報告** | 生成自動化的風險總結，供審核員參考 | Medium |

## 3. ESG 評分框架邏輯
系統採用「風險曝露 (Exposure) × 風險管理 (Management)」框架：
- **風險曝露 (Exposure) - 權重 60%**: 碳排強度、產業敏感度、資源依賴度。
- **風險管理 (Management) - 權重 40%**: 漂綠爭議、ISO 認證、轉型計畫。

## 4. 使用者流程
1. **導入資料**: 上傳企業 CSR 報告或新聞。
2. **AI 分析**: 自動解析並計算風險。
3. **評分輸出**: 顯示分數與風險等級 (AAA-CCC)。
4. **決策輔助**: 提供 AI 評分理據 (Reasoning)。
